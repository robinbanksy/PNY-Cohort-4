Block Display

This is a paragraph inside which a block displayed element is placed. The block element breaks the flow of the text.

Inline Display

This is a paragraph inside which an inline displayed element is placed. The inline element does not break the flow of the text.

Inline-Block Display

This is a paragraph inside which an inline-block displayed element is placed. The inline-block element does not break the flow of the text, but it can have a width and height set.

Display: None

This is a paragraph inside which an [element with display set to none] is placed. The element with display set to none will not be visible on the page, and will not take up any space.
