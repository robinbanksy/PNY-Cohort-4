Complex CSS Selectors

As covered in the previous lesson, CSS rules take the following form:

selector {
    property: value;
}

The "selector" part of a CSS rule is what tells the browser which HTML elements to apply the rule to. We've seen that we can use element type selectors, and also class selectors and id selectors to target specific elements. Today we'll see examples of more complex selectors, which are often composed of multiple simple selectors.

Multi-Class Selectors

<div class="box red">Red Box</div>
<div class="box blue">Blue Box</div>
<div class="box green">Green Box</div>

.box.red {
    background-color: red;
}

Remember that a single element can have multiple classes applied to it. Above we can see a CSS rule that targets elements only if they have both the classes "box" and "red".

Descendant Selectors

<div class="container">
    <div class="item">Item 1</div>
    <div class="item">Item 2</div>
    <div class="item">Item 3</div>
</div>
<div class="item">Item 4</div>
<div class="item">Item 5</div>

.container .item {
    color: red;
}

This rule will apply to any element with the class "item" that is a descendant of an element with the class "container". In other words, it will apply to any element that has the class "item" and is inside an element with the class "container". Notice that this rule will not apply to the "item" elements outside of the "container" element.

Child Selectors

<div class="container">
    <div class="item">
        <div class="item">Item 4</div>
    </div>
    <div class="item">Item 2</div>
    <div class="item">Item 3</div>
</div>

.container > .item {
    font-size: 20px;
}

This rule is similar to the descendant selector, but it only applies to elements that are direct children of the "container" element. In other words, it will apply to any element with the class "item" that is a direct child of an element with the class "container". Notice that this rule will not apply to the "item" elements that are nested inside other elements within the "container" element.

Other Examples

h2#section-one {
    letter-spacing: 2px;
}

This rule targets an h2 element that also has the id "section-one".

.nav-button#home-button {
    background-color: blue;
}

This rule targets an element with the class "nav-button" that also has the id "home-button".

header img.logo {
    width: 20px;
    height: 20px;
}

This rule targets any img element with the class "logo" that is a descendant of a header element.
