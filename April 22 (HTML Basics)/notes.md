# What is Git?

- Git is a version control system.
- It keeps track of changes to a project and allows us to revert to earlier versions.
- Git also makes collaboration easier.

## How Git works:
- Create a Git repository, which is a folder for project files.
- Git runs inside the repository and tracks file history.

# What is GitHub?

- GitHub is an online hosting service for Git repositories.
- A repository copy stored on GitHub is called a remote repository.
- A repository copy on your device is called a local repository.
- Git commands sync changes between the local and remote copies.

## Common Git/Github workflow:
1. Pull changes from the remote repository on GitHub to your local repository.
2. Make changes to code in the local repository.
3. Push those changes back to the remote repository on GitHub.

# GitHub Codespaces

- GitHub Codespaces provides a browser-based version of VS Code.
- It lets you use an editor without installing it locally.
- Codespaces are created on a branch of a repository and act like a local workspace.

# HTML Attributes

- Attributes add extra information to HTML elements.
- They appear inside the opening tag and have a name and a value.
- Example attributes include href and target.

## Example HTML with attributes:
- `<h2 id="section-one-heading">Section Title</h2>`
- `<a href="https://example.com" target="_blank">Link</a>`

# HTML IDs

- An id is a special HTML attribute that uniquely identifies an element.
- IDs allow CSS and JavaScript to target specific elements.
- Two elements cannot share the same id on one page.

# The Anchor Tag

- The anchor tag `<a>` creates hyperlinks in HTML.
- It can link to other pages or sections on the same page.
- Use `href` to specify the destination.

## Examples:
- `<a href="about.html">About Us</a>` links to another page.
- `<a href="#section-one">Go to Section One</a>` links to a section on the same page.

# The Image Tag

- The image tag `<img>` displays images in HTML.
- It is an empty tag and does not require a closing tag.
- Use the `src` attribute to specify the image source.
- Use the `alt` attribute to provide a text description for accessibility.

## Example image tag:
- `<img src="https://example.com/image.jpg" alt="Description of image">`

- The `alt` description is read by screen readers and shown if the image fails to load.
