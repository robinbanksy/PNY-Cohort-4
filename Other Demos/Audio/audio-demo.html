Audio Element With Controls

Has the autoplay and controls options included.

Most browsers automatically block any non-muted autoplay media, so most likely, your autoplay audio element won't work by itself. You can add the muted option, which will allow it to autoplay, and then allow the user to unmute it if desired, or simply leave the controls such that the user can start playback themselves.
