What's a Shift Cipher?

A shift cipher is a simple encryption technique where each letter in the original message is "shifted" by a certain number of positions in the alphabet. For example, in a cipher with a shift of 3, 'A' would become 'D', 'B' would become 'E', and so on. If we reach the end of the alphabet, we wrap around to the beginning, so 'Z' would become 'C'.

Rot13

Rot13 is a specific type of shift cipher where each letter is shifted by 13 positions. This means that 'A' becomes 'N', 'B' becomes 'O', and so on. Rot13 is special, in that it's its own inverse. This means that if you apply Rot13 to a message twice, you will get back the original message. For example, if you apply Rot13 to "Hello, World!", you would get "Uryyb, Jbeyq!". If you apply Rot13 again to "Uryyb, Jbeyq!", you would get back "Hello, World!". This is because the shift number of the cipher is 13, which is exactly half of 26, the number of letters in the English alphabet.

So, in other words, unlike most shift ciphers, where we follow one procedure to encrypt our message, and a different procedure to decrypt it, Rot13 uses the same procedure for both encryption and decryption.
